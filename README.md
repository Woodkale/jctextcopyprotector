jctextcopyprotector
===================

This plugin will give some protection to your content being getting copied. There are many options that gives you granular control over plugin functionality.


<h2>Options</h2>
<ul>
	<li><strong>blockRightClick:true|false</strong>,
This option if set to true will block context menu to appear</li>
	<li><strong>blockDocTextSelection:true|false,</strong>
This option if set to true will disable text selection using mouse</li>
	<li><strong>useCSS:true,
</strong>This option if set to true will use CSS to block text selection on your document</li>
	<li><strong>blockPageSave:true,
</strong>This option if set to true will block keyboard event such as Ctrl+A, Ctrl+S , Ctrl+C</li>
	<li><strong>alertUser:false,
</strong>This option if set to true will show an alert message when a blocked event is detected.</li>
	<li><strong>alertMessage:''
</strong>Custom alert message</li>
</ul>
